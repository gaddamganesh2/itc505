Lights Out Game

Addendum: Development Process and Creative Choices

  1. Game Structure and Initialization:
    • Goal: To create a Lights Out game using HTML, CSS, and JavaScript.
    • Process:
      • Defined the game structure using HTML with a game board and buttons.
      • Used CSS for styling, including animations and transitions.
      • JavaScript was employed for game logic, such as toggling lights and checking for a win condition.
  2. Game Initialization (`script.js`):
    • Challenges:
      • Ensuring the creation of a dynamic game board with toggleable lights.
    • Solutions:
      • Used nested loops to generate the grid of lights dynamically.
      • Assigned event listeners to each light for toggling.
  3. Random Initial State:
    • Creative Choices:
      • Implemented a function (`setRandomInitialState`) to randomly turn lights on or off at the start of the game for variety.
      • Used the `Math.random()` function for a balanced distribution.
  4. Toggle Lights and Win Conditions:
    • Creative Choices:
      • Developed the `toggleLights` function to switch the state of clicked lights and adjacent lights.
      • Integrated the `checkWin` function to determine if all lights are turned off.
      • Added a congratulatory message and reset the game on a win.
  5. Styling (`styles.css`):
    • Creative Choices:
      • Applied a flex layout for responsiveness.
      • Incorporated hover effects on lights and buttons for a visually appealing interface.
      • Animated lights with a bounce effect on hover.
  6. Game Controls (`index.html`):
    • Creative Choices:
      • Included "Start," "New Game," and "Exit Game" buttons.
      • Created an "Addendum" button to showcase additional information.
  7. Addendum and UI Changes:
    • Creative Choices:
      • Added an exit button for a cleaner user interface.
      • Altered font and background image for aesthetic improvements.
      • Implemented a pulse animation on the exit button for user engagement.
